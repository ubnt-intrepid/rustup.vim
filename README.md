# rustup.vim
A thin wrapper for calling [`rustup`](https://github.com/rust-lang-nursery/rustup.rs) from Vim

## Current status

* `rustup#toolchain#list()`
* `rustup#toolchain#default()`
* `rustup#show()`

Other utility functions:

* `rustup#source_path()`

## Installation

### vim-plug

```vim
Plug 'ubnt-intrepid/rustup.vim'
```
